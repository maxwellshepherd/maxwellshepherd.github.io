# Maxwell Shepherd - Portfolio

Personal portfolio website built with React, Vite, and Tailwind CSS.

## 🌐 Live Site

Visit: [https://maxwellshepherd.github.io](https://maxwellshepherd.github.io)

## 🚀 Technologies

- **React** - UI framework
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **GitHub Pages** - Hosting and deployment

## 💼 About

Portfolio showcasing Maxwell Shepherd's work as an Industrial Engineer and Data Scientist at Boeing, currently pursuing an M.S. in Data Science at Johns Hopkins University.

## 🛠️ Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📦 Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the main branch via GitHub Actions.

Manual deployment:
```bash
npm run deploy
```
